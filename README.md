# 学生成绩管理系统

```
基于 vue + iview 开发的学生成绩管理系统
```
[基于 node + express 开发的后台](https://github.com/hitore/stdsysapi)

## 拉取仓库
```
git clone git@github.com:hitore/stdsys.git
cd stdsys
```

## 安装依赖
```
npm install
or
cnpm install
or 
yarn install
```

## 本地开发
```
npm run dev
```

## 打包
```
npm run build
```
